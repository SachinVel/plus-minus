Are pdf formats supported?
No, currently BSA does not support .pdf format bank statements to be imported.
Can we convert a .pdf type bank statement into .xlsx format and import into BSA?
Yes, that is allowed, although, it has come to our attention that, in most occasions, PDF statements converted into excel usually contain more than one sheet and images in the file which make it extreme difficult for us to process the data and give out a desired output. PDF is meant for only data transfer and not data analysis. So, the converted statements can be brought to the standard bank statement format by the user and uploaded for accurate result.
Note: If there are watermarks, images, seal & signatures in the converted file, kindly remove the same before importing the .xlsx file.
Can we upload a bank statement in which we have worked & applied formulae to certain cells?
Yes, that is allowed, although it is recommended to upload the statements downloaded from the bank's internet banking website without making any modifications (or) performing any arithmetic functions to the aforementioned file.

This is just an illustrative format to help the user to get a better picture which in turn helps us to serve you better.) What does the error message "File Content is invalid" mean? Kindly check your file for inputs like
  • ********** in between the transactions (or) at the start (or) end of the data fields
  • Invalid date format (for instance - date is entered as a text)
  • Invalid amount format (for instance - amount is entered as a text)
What is the order in which the input fields need to be selected?
The input fields are to be selected in the following order (by whatever name called in the bank statement in question):
  1. Date
  2. Description
  3. Debit
  4. Credit
  5. Balance
What recourse do I have if I have selected a wrong cell in the input field?
If the user has selected the wrong cell as the input field, the X button near the input field can used to clear the value fed into the input field and another cell can be selected in its place.
What does it mean if the cell values display [object Object] in the preview page of the uploaded file?
The error [object Object] appears when the format of the data in the cells are not suitable. Kindly verify the format of the excel sheet again using the pointers given in optimal bank statement format and upload the file again. If the error still persists, kindly contact our support team.
Can I move records from receipts to payments and vice versa?
No, BSA not does allow movement of transactions from Receipts to Payments & vice versa.
Will the exported file contain a list of all the transactions under a certain head?
No. The exported file shall contain only a summary of the Bank Analysis finalised by the user in the Consolidation page. The same can be viewed by the user only in the Consolidation page & not the resultant .xlsx file.
